## Задача 1 ##
## Условие задачи ##
Дано трехзначное число, указать совпадающие разряды. Программа должна выводить диагностическое сообщение вида "число единиц и десятков равно".
## 1. Алгоритм блок-схема ##
### Алгоритм ###
1. Начало
2. Инициализируем переменные ```a```, ```x```, ```y```, ```z```.
3. Пригласить к вводу значения переменной ```a```.
4. Присваиваем переменным значения:
    * ```x``` = ```a``` / 100
    * ```y``` = (```a``` / 10) % 10
    * ```z``` = ```a``` % 10
5. Выполняем проверку условий: если число сотен (```x```) равно числу десятков (```y```), то выводим диагностическое сообщение "число сотен и десятков равно". Аналогично с сотнями-единицами и десятками-единицами.
6. Конец.
### Блок-схема ###
![](https://github.com/kiri4art/Lab6/blob/main/lab6task1.png)
## 2. Реализация программы ##
```
#define _CRT_SECURE_NO_DEPRECATE
#include <stdio.h>
#include <locale.h>
#include <math.h>

int a, x, y, z;
// a - число; x, y, z - сотни, десятки, единицы соответственно

void main()
{
	setlocale(LC_ALL, "RUS");
	puts("Введите трёх-значное число:");
	scanf("%d", &a);
	x = a / 100;
	y = (a / 10) % 10;
	z = a % 10;
	if (x == y)
		printf("число сотен и десятков равно\n");
	if (x == z)
		printf("число сотен и единиц равно\n");
	if (z == y)
		printf("число единиц и десятков равно\n");
}
```

Составить программу, которая проверяет принадлежит ли заданная пользователем точка с координатами (х, у) заданным закрашенным (заштрихованным) областям, показанным на рисунках в табл., и выводит на экран сообщение.

## 1. Алгоритм блок-схема ##
### Алгоритм ###
1. Начало
2. Инициализируем переменные ```x```, ```y```
3. Пригласить к вводу координат ```x```, ```y```
4. Проверяем, входит ли точка с введенными координатами в пределы заданной области, после чего выводим результат на экран
5. Конец

## 2. Реализация программы ##
```
#define _CRT_SECURE_NO_DEPRECATE
#include <stdio.h>
#include <locale.h>
#include <math.h>

int x, y;

void main()
{
	setlocale(LC_ALL, "RUS");
	puts("Введите координаты через пробел:");
	scanf("%d %d", &x, &y);
	(((-2 <= x <= 5) && ((y == 3) || (y == 2))) || ((-2 <= x <= 3) && (2 <= y <= 5))) ? printf("Точка с координатами (%d, %d) находится в области!", x, y) : printf("Точка с координатами (%d, %d) не находится в области!", x, y);
}
```
